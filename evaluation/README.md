evaluation/

This folder contains performance measurement utilities.

performance_analyzer.py:
Computes key portfolio metrics including cumulative return, Sharpe ratio, volatility, and maximum drawdown.

generate_contest_submission.py:
Creates the final submission files required for the SecureFinAI Task 5 evaluation by filtering logs within the official evaluation window (Apr 20 – May 1).

These scripts ensure reproducible and standardized evaluation of trading performance.
